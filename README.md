          # Étape 7 – Quiz & réflexions

  Question : Que renvoie A + B si A fait (4,) et B fait (3,) ?
Réponse : Erreur (ValueError), shapes incompatibles.
Question : Quelle est la différence entre np.dot(A, B) et A * B ?
Réponse : np.dot fait un produit scalaire ou matriciel.
          A * B fait une multiplication élément par élément.
Question : Quelle est la shape de M + vecteur_ligne ?
Réponse : (3, 3)
Question : Comment filtrer des valeurs négatives après un np.log ?
Réponse : filtered = log_vals[log_vals >= 0]
